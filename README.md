# Homilies on the Beginning of the Holy Forty Days  
## St. John Chrysostom — Modern English Translation

This repository contains a modern English translation of  
**St. John Chrysostom’s Homily on the Beginning of the Holy Forty Days (Lent)**.

### Translation Note
This translation was produced using the Greek text of
St. John Chrysostom as found in *Patrologia Graeca*, volumes 49–53,
with comparison to historical Latin witnesses.

The translation was generated with the assistance of ChatGPT
and reviewed for clarity and faithfulness to the Greek text.
It is intended as a readable modern-English rendering,
not as a critical scholarly edition.

### Public Domain Status
This English translation has been explicitly released into the public domain
under the CC0 1.0 Universal dedication.
